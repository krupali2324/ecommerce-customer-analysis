# 📊 E-commerce Customer Purchase & Retention Analysis

## 📌 Overview
This project analyzes customer behavior, retention, and revenue trends using SQL and Power BI.

## 🛠 Tools Used
- SQL (MySQL)
- Power BI
- Excel

## 📂 Dataset
- 500 customers
- 1000 orders
- 3000+ order items

## 🔍 Key Insights
- Repeat customers generate higher revenue
- Top customers contribute a large portion of sales
- Certain product categories dominate sales
- Revenue trends show consistent patterns
